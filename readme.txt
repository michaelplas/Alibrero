1. Verzamel een lijst van Albireo IP adressen die je wil updaten en zet deze in iplistt.txt
2. Voeg het ip van je tftp server toe aan tftp.txt (vaak je eigen ip). LET OP: kan geen localhost zijn.
3. Voer setup.py uit en voer het aantal scripts in die je wil gaan runnen (meestal een stuk of 4 - 7 per laptop)
4. Er zijn nu aan aantal mappen gemaakt, dit zijn de aangemaakte updaters. Voer van elke updater de start.py uit.
5. Profit.
